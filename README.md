# ⚡ LinkedIn Easy Apply Automation Bot

A browser-based automation script that helps speed up the LinkedIn Easy Apply process by automatically filling and navigating job applications based on predefined user details.

---

## 💡 Why This Project Matters

Applying to jobs manually can be repetitive and time-consuming, especially when many postings require the same information.

This project was built to reduce repetitive effort by automating parts of the application workflow while still allowing the user to review and control the process.

---

## 🚀 Key Features

- Automates LinkedIn Easy Apply workflows  
- Fills common application fields  
- Speeds up repetitive job applications  
- Runs directly in the browser using JavaScript  
- Improves application efficiency  
- Supports customizable user inputs  

---

## 🛠️ Tech Stack

## ⚙️ How to Use

### 1. Open LinkedIn Jobs

Go to LinkedIn and search for jobs using filters like:

Easy Apply
Location
Experience level
Job title

### 2. Open an Easy Apply Job

Click on a job posting that supports Easy Apply.

### 3. Open Browser Developer Tools

Use:

Right Click → Inspect → Console

or:

Ctrl + Shift + J

### 4. Paste the Script

Copy the JavaScript code from this repository and paste it into the browser console.

### 5. Run the Automation

The script will attempt to:

Fill repetitive fields

Navigate application steps

👉 You should still review before submitting
